Welkom bij de angular lite lesapplicatie.
In deze applicatie wordt de volledige les van Angular Lite uitgelegd, stap voor stap.

Deze applicatie is ontworpen door Quinten Van Casteren voor de les Angular Lite.
Het doel van deze applicatie is de volledige lesinhoud van Angular in een eenvoudige applicatie gieten die de docent Michael Cloots dan kan gebruiken om zijn lessen te vereenvoudigen.
De applicatie geeft uitleg over alle nodige code en gebruikt deze ook zelf om voorbeelden te tonen.

De applicatie maakt gebruik van twee kleuren (blauw en groen) om code aan te duiden.
Groen is voor in de console, blauw voor in de applicatie zelf.
