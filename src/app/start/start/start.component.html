Getting Started

Tijdens dit hoofdstuk zullen wij bekijken hoe we een angular applicatie aanmaken.

Project aanmaken

We beginnen installeren van de Angular CLI via het commando npm install -g @angular/cli@latest.
Bij versie problemen volg de volgende stappen:

  • npm uninstall â€“g @angular/cli
  • npm cache verify
  • npm install -g @angular/cli@latest

Vervolgens gaan we naar de map dat we een nieuw project willen aanmaken en gebruiken we het commando ng new HelloWorldAngular6.

  • ng: Angular CLI
  • new: Nieuwe applicatie
  • HelloWorldAngular6: zelfgekozen naam

Ten slotte voeren we het commando ng serve uit om onze applicatie op te starten. Surf hierna naar http://localhost:4200/ om de applicatie na te kijken.

Eerste blik op Angular

  • package.json: configuratie van de applicatie en scripts die men kan uitvoeren
  • angular.json: configuratie i.v.m. builden, testen en omgevingen
  • tsconfig.json: typescript compiler
  • e2e: alles rond end-to-end testing
    • ng e2e om end-to-end testen uit te voeren
  • node_modules: alle npm packages
  • src: hieronder bevind zich de applicatie en zal je het meeste werken
