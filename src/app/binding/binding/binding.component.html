Databinding

Tijdens dit hoofdstuk zullen wij kennis maken met verschillende vormen van data binding.

Interpollation

Een eerste vorm van databinding is Interpollation. Dit hebben wij vorig hoofdstuk al deels gezien. Bij interpollation halen we data op vanuit onze ts file via {{...}}. Dit is de eenvoudigste manier om data op te halen vanuit je applicatie.

Property Binding

Bij property binding gaan we ongeveer hetzelfde doen als bij interpollation. Alleen wordt de data nu als property gebruikt.
Een voorbeeld hiervan is de onderstaande afbeelding:
<img [src]='imageUrl' [alt]='imageAlt' />
ImageAlt en ImageUrl zijn hierbij al ingegeven in de ts file op dezelfde manier als bij interpollation.

Event Binding

Bij event binding gaan we een actie aanmaken op bijvoorbeeld een button die dan iets zal laten gebeuren in onze component.
Om te beginnen maken we een event aan in onze component via de voorbeeldcode:
onButtonClick() {
alert('Ya wanted an alert, mate;')
}
 Hierna maken we deze button aan via deze voorbeeldcode:
<button (click)="onButtonClick()">Alert?</button>
