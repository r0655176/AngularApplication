Tijdens dit hoofdstuk zullen wij nagivatie in onze applicatie maken.

Routes

In een applicatie moet er natuurlijk navigatie mogelijk zijn. Links op deze pagina staat er bijvoorbeeld een navigatiebalk waarmee we op deze pagina komen. Dit kan in angular gemakkelijker via de @angular/router library.
Voordat we starten maken we 3 nieuwe componenten aan: Home, Contact en Help.

Om te beginnen moeten we onze app module deze library importeren.
Dit doen we via deze code:
import { Routes, RouterModule } from '@angular/router';

Hierna voegen we routes toe de we linken naar onze verschillende componenten.
const appRoutes: Routes = [
{ path: '', component:HomeComponent },
{ path: 'contact', component:ContactComponent },
{ path: 'help', component:HelpComponent },
];

Ten slotte importeren we onder imports de routermodule met onze zelfgemaakte routes.
RouterModule.forRoot(appRoutes, {enableTracing: true})

Routing

Nu we alle routes hebben gelegt kunnen we eenvoudig deze tonen.
Via <router-outlet></router-outlet> kunnen we de component dat we in de routes hebben meegegeven tonen. Op de startpagina is dit Home.
Hierna maken we links naar deze aan via deze code:
<a routerLink="/" routerLinkActive="active">Home</a>
<a routerLink="/contact" routerLinkActive="active">Contact</a>
<a routerLink="/help" routerLinkActive="active">Help</a>

De property routerLinkActive zorgt ervoor dat we kunnen zien of er op deze link geklikt is geweest.
