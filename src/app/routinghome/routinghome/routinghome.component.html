Routing

Tijdens dit hoofdstuk zullen wij nagivatie in onze applicatie maken.

Routes

In een applicatie moet er natuurlijk navigatie mogelijk zijn. Links op deze pagina staat er bijvoorbeeld een navigatiebalk waarmee we op deze pagina komen. Dit kan in angular gemakkelijker via de @angular/router library.
Voordat we starten maken we 3 nieuwe componenten aan: Home, Contact en Help.

Om te beginnen moeten we onze app module deze library importeren in onze appModule.
Dit doen we via deze code:
import { Routes, RouterModule } from '@angular/router';

Sidenote: In deze applicatie is alle code toegevoegd aan routing.module.ts en shared.module.ts in plaats van app.module.ts. De reden hiervoor zien we in een later hoofdstuk. Ook zijn de voorbeelden niet toegepast in deze applicatie voor dezelfde reden.

Hierna voegen we routes toe de we linken naar onze verschillende componenten.
const appRoutes: Routes = [
{ path: '', component:HomeComponent },
{ path: 'contact', component:ContactComponent },
{ path: 'help', component:HelpComponent },
];

Ten slotte importeren we onder imports de routermodule met onze zelfgemaakte routes.
RouterModule.forRoot(appRoutes, {enableTracing: true})

Routing

Nu we alle routes hebben gelegt kunnen we eenvoudig deze tonen.
Via <router-outlet></router-outlet> kunnen we de component dat we in de routes hebben meegegeven tonen. Op de startpagina is dit Home.
Hierna maken we links naar deze aan via deze code:
<a routerLink="/" routerLinkActive="active">Home</a>
<a routerLink="/contact" routerLinkActive="active">Contact</a>
<a routerLink="/help" routerLinkActive="active">Help</a>

De property routerLinkActive zorgt ervoor dat we kunnen zien of er op deze link geklikt is geweest.

Angular Material

Via Angular Material kunnen we designcomponenten toevoegen aan onze applicatie.
Wij gebruiken dit in deze applicatie voor de sidenav te maken.
Om te beginnen moeten we alle libraries installeren via npm install --save @angular/material @angular/cdk @angular/animations

Vervolgens importeren we deze toe via de onderstaande code en voegen we deze toe aan de imports van appModule (en alle andere componenten waar we deze willen gebruiken).
import { MatSidenavModule, MatListModule, MatCardModule, MatButtonModule, MatToolbarModule } from '@angular/material';

Ten slotte kunnen we in onze html een sidenav aanmaken. Dit doen we via deze code:
<mat-sidenav-container class="example-container">
<mat-sidenav #sidenav mode="side" opened="true">
<mat-nav-list>
<a mat-list-item routerLink="/">Home>/a>
<a mat-list-item routerLink="/contact">Contact</a>
<a mat-list-item routerLink="/help">Help</a>
</mat-nav-list>
</mat-sidenav>
<mat-sidenav-content>
<router-outlet></router-outlet>
</mat-sidenav-content>
</mat-sidenav-container>

Style

Een handige manier om een applicatie te verbeteren is om het een style te geven. De gemakkelijkste manier voor dit te doen is door aan styles.css een standaarstyle toe te voegen. Voeg onderstaande code toe. Dit is een standaardstyle van angular material zelf.
@import "~@angular/material/prebuilt-themes/indigo-pink.css";

Ook kunnen we een style aanmaken in de css file van elke container. Een voorbeeld hiervan is de sidenav over heel de pagina de volledige hoogte maken. Dit doen we via de volgende code:
.example-container {
position: absolute;
top: 0;
bottom: 0;
left: 0;
right: 0;
}

Ook is het mogelijk een eigen theme te maken. Hiervoor moet je eerst een nieuw bestand theme.scss aanmaken en aan angular.json onder styles toevoegen. Ook mag er geen style meer in styles.css staan. In theme.scss. voegen we eerst deze code toe:
@import '~@angular/material/theming';
@include mat-core();
 Hiermee voegen we theming toe zodat we zelf themes kunnen maken en mat-core, waar de belangrijkste styles al in zitten.

Hierna kunnen we zelf kleurpaletten toevoegen via mat-palette:
$helloworld-app-primary: mat-palette($mat-teal);
$helloworld-app-accent: mat-palette($mat-amber, A200);
$helloworld-app-warning: mat-palette($mat-red, 600);
$helloworld-app-secondary: #607D8B;

Ten slotte bundelen we deze paletten in 1 theme en includen we deze in heel de applicatie:
$helloworld-app-theme: mat-light-theme($helloworld-app-primary, $helloworld-app-accent, $helloworld-app-warning, );
@include angular-material-theme($helloworld-app-theme);
