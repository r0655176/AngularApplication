New Component: {{title}}

Nu we de begrijpen hoe een angular component werkt gaan we een nieuw component aanmaken en we noemen het directive. Dit doen we via ng generate component directive.
Deze component wordt dan normaal toegevoegt aan app.module.ts als declaration. Dit betekend dat het directive component nu in heel de applicatie gemaakt kan worden.
Ten slotte voegen we in app.component.html de code <app-directives></app-directives> toe. Deze zorgt ervoor dat de view van directive wordt toegevoegt op die locatie in het html bestand.

Sidenote: In deze applicatie is alle code toegevoegd aan anatomie.module.ts en anatomie.component.html in plaats van app.module.ts en app.component.html. De reden hiervoor zien we in een later hoofdstuk. Alle code van dit onderdeel bevind zich in de component directive, zowel in deze applicatie als je eigen applicatie.

Om te beginnen moeten we leren hoe we data uit ons ts bestand naar html kunnen brengen. Dit is vrij eenvoudig.
We starten met het toevoegen van een title aan het ts bestand van directive net zoals er bij app.component.ts al gedaan was.
Hierna voegen we deze toe aan onze text in html via {{title}}.

In angular kan men ook een ngClass aanmaken om hier dan verschillende ccs-eigenschappen aan te geven.
Dit doet men via het attribuut [ngClass] toe te voegen aan bijvoorbeeld deze p-tag.
Hierna voeg je in directive.ts ook deze klasse toe samen met de classen aangemaakt in de css file.
Bijvoorbeeld in deze p-tag is er gebruik gemaakt van: klasses: string [] = ['blue','bold'];

Een volgende mogelijkheid van angular is if-statements te gebruiken op verschillende klassen via [ngIf].
Een voorbeeld hiervan is: <div *ngIf="isNotLoremIpsum">Voorbeeld</div>
Hierna voegt men in directive.ts code toe voor isLoremIpsum true of false te maken:
isLoremIpsum: boolean = true;

Deze text is een voorbeeld hiervan.

Ipsum Lorem

Ook is er de mogelijkheid een else toe te voegen. Dit kan via ng-template.
De code hierboven is hier al een voorbeeld van en is gecodeert via deze text:
<div *ngIf="isNotLoremIpsum; else anderTemplate">Deze text is een voorbeeld hiervan.</div>
<ng-template #anderTemplate>
<p [ngClass]="klasses">Ipsum Lorem</p>
</ng-template>

Vervolgens kunnen we ook angular gebruiken voor lijsten te maken van zelf ingegeven gegevens.
Zo hebben wij bevoorbeeld in directive.ts deze lijst: frameworks: string[] = ["Angular", "React", "Vue"];
Hieronder maken we hieronder dan een lijst van deze frameworks via ngFor:
<ol><li *ngFor= "let frame of frameworks">{{frame}}</li></ol>

  1. {{frame}}

Hiermee kunnen we leuke dingen doen zoals dit:

{{color}}

Probeer dit nu zelf. De oplossing vind je in de code van deze applicatie als je ze kan vinden :P

Ten slotte kan je via angular ook een switch maken via [ngSwitch]
In directive.ts maak je dan een element aan, bijvoorbeeld score.
Daarna maak je de onderstaande switch aan via onderstaande code:
<div [ngSwitch]="score">
<p *ngSwitchCase="'0'">Oef, je score is 0</p>
<p *ngSwitchCase="'50'">Oef, je score is 50: net geslaagd!</p>
<p *ngSwitchCase="'100'">Super, je score is 100: prachtig gedaan!!</p>
<p *ngSwitchDefault>Je score is {{score}}.</p>
</div>
