Nood aan typescript

Javascript is een geweldige taal, maar heeft enkele nadelen. Deze nadelen zorgen ervoor dat een applicatie op pure Javascript soms vrij ingewikkeld wordt. Hierom gebruiken wij bij Angular een combinatie van Javascript en Typescript.
De voornaamste nadelen van Javascript zijn:

  • Grote projecten lezen als spaghetticode
  • Geen data types
  • Scope bepalen van stukken code is intewikkeld

Hierom hebben wij een taal nodig die we kunnen transcompilen naar javascript.
De oplossing: Typescript
