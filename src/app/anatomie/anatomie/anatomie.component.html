Angular Anatomie

Een Angular applicatie bestaat uit een mengeling van meerdere componenten die wij in de komende hoofdstukken gaan opbouwen.
Deze componenten bestaan dan weer uit 3 delen: Een template, een class en metadata.
Ga naar app.component.ts en zie het volgende:
import { Component } from '@angular/core';
@Component({
selector: 'app-root’,
templateUrl: './app.component.html’,
styleUrls: ['./app.component.css']
})
export class AppComponent {
title = 'HelloWorldAngular6';
}

In de typescipt file staan zowel de metadata en de class properties.
De metadata toont aan dat dit een component is en linkt de component aan het template (de html en css files).
De class properties zijn de verschillende properties waarmee de component zal werken. Standaard zit hier de title al bij.
Het template bepaalt hoe het component er uit ziet. De html en css files hiervan worden in de metadata gelinkt.
